# Diabetic Readmission Prediction System

## Project Overview
This project predicts whether a diabetic patient is likely to be readmitted to the hospital using machine learning algorithms.

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Machine Learning Model
- Prediction of patient readmission
- Model evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Project Structure
app/
models/
notebooks/
reports/
src/

## Installation

git clone <repository-link>

cd Diabetic-Readmission-Prediction-System

pip install -r requirements.txt

python run.py
